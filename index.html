Color de fondo

La Background-color es la propiedad especifica del color de fondo del elemento

Ejemplo: background-color: #fff;

Opacidad/Transparencia

La opacity propiedad especifica/transferencia de un elemento. Puede tomar un valor de 0,0 a 1,0. Cuando menor sea el valor, mas transparente:

Ejemplo:

div{
background-color: red;
opacity: 0.3;
}

Repeticion de Imagen de Fondo

De forma predeterminada, la background-image propiedad repite una imagen tanto horizontal como verticalmente.
